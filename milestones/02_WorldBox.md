# 02. WorldBox (沙盒上帝模拟): 传统社会模拟与大模型智能体桥梁的交汇

## 基础信息
* **时间**：2012年（最初发布，后持续演进并于2025/2026年接入大模型/MCP）
* **来源**：
  * **游戏官网**：[Super WorldBox](https://www.superworldbox.com/)
  * **开源桥梁**：[worldbox-mcp](https://github.com/fullya99/worldbox-mcp) (Model Context Protocol 智能体控制桥梁)

---

## 核心架构与技术原理

WorldBox 是一款极具代表性的二维像素风上帝模拟沙盒游戏。在 LLM 爆发前，它已经具备了极其深度的自主社会与文明演进机制。而在 2025/2026 年，通过开源的 `worldbox-mcp` 桥梁，它正式演变为大模型智能体的具身社会模拟环境。其核心架构包含：

1. **自主文明演进系统（Civilization Mechanics）**：
   * **四大种族与地缘政治**：模拟人类、精灵、矮人、兽人四大种族的自主繁衍、资源采集（木材、石料、黄金）、城镇建设与扩张。
   * **领袖与领主机制（Lords & Kings）**：每个王国由国王领导，每个村庄由领主管理。他们拥有独特的个人属性（外交、管理、军事），这些属性动态决定了王国的扩张倾向、内政效率、外交关系（结盟或宣战）以及村庄的忠诚度。
   * **文化传承系统（Culture）**：文化是绑定在个体身上的知识网络。随着时间推移，文明会自主研发科技（如造船、采矿、武器制造），解锁新的社会行为。

2. **纪元与自适应演化（Ages System）**：
   * 引入“纪元系统”（如希望纪元、绝望纪元、灰烬纪元），改变全球环境、食物产量和怪物威胁，迫使各个文明进行自适应调整，展现出极强的环境适应与社会韧性。

3. **大模型智能体接入层（worldbox-mcp Bridge）**：
   * **MCP 协议打通**：通过 `WorldBoxBridge` Mod，将游戏内部状态与控制权通过 Model Context Protocol (MCP) 暴露给外部 LLM Agent。
   * **感知-决策闭环**：AI 智能体可以调用 26 个 MCP 工具（如 `worldbox_spawn` 刷怪、`worldbox_paint_tile` 改变地形、`worldbox_recv_messages` 接收消息），实现完整的 Agentic Loop。
   * **多 Agent 混合自治**：支持多个 AI 智能体在同一个世界中扮演不同角色（如神明、阵营领袖、观察者、旁白），通过内置的消息总线进行跨 Agent 沟通、地缘对抗或合作。

## 关键贡献与对社会模拟的启示

* **沙盒社会模拟的工程典范**：在没有大模型语义理解的时代，WorldBox 仅凭精妙的微观数值规则和状态机，就成功复现了极其逼真且不可预测的帝国兴衰、地缘政治博弈、文化扩散和历史变迁，证明了规则驱动模拟的上限。
* **传统游戏向 AI Agent 物理具身测试床的跃迁**：通过 `worldbox-mcp`，WorldBox 成功从一个“纯观察/手动干预”的传统沙盒游戏，转变为一个支持大模型智能体实时生存、决策和多 Agent 社交互动的**物理具身社会模拟平台**。
* **混合自治与多智能体博弈的新范式**：展示了人类玩家、规则驱动的 NPC、以及大模型驱动的 Agent 可以在同一个物理世界中无缝共存、博弈与协作，为未来的混合现实社会和 AGI 具身智能测试提供了极佳的范例。
