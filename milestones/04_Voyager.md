## 核心架构与技术原理

Voyager 是第一个在《我的世界》（Minecraft）中实现**无监督、终身学习（Lifelong Learning）**的具身智能体。它没有采用传统的强化学习梯度更新，而是完全基于大语言模型（GPT-4）进行代码级别的决策。其核心由三大模块组成：

```
                    ┌────────────────────────────────┐
                    │       Automatic Curriculum     │ (自主提出探索目标)
                    └──────────────┬─────────────────┘
                                   │ (Task)
                                   ▼
                    ┌────────────────────────────────┐
             ┌─────►│         Skill Library          │◄─────┐
             │      └──────────────┬─────────────────┘      │
             │                     │ (Retrieve/Generate)    │
             │                     ▼                        │
             │      ┌────────────────────────────────┐      │
             │      │  Action Agent (Code Execution) │      │
             │      └──────────────┬─────────────────┘      │
             │                     │ (Execute)              │
             │                     ▼                        │
             │      ┌────────────────────────────────┐      │
             │      │      Environment Feedback      │      │
             │      └──────────────┬─────────────────┘      │
             │                     │ (Error/State)          │
             └─────────────────────┴────────────────────────┘
```

### 1. 自动课程（Automatic Curriculum）
* 传统的社会模拟中，Agent 的目标往往是静态或人类预设的。
* Voyager 的自动课程模块会根据 Agent 当前的物品背包、生命值、所处地形等状态，自主提出“由易到难”的探索任务（例如：从“撸树获取木头”到“制作石镐”，再到“击杀僵尸”）。

### 2. 技能库（Skill Library）—— “代码即技能（Code as Skills）”
* **痛点**：如果每次行动都让 LLM 输出一堆自然语言或零散命令，不仅 token 消耗极大，而且难以维持长期的复杂操作。
* **解决方案**：Voyager 将成功的行动序列固化为 **JavaScript 代码函数**。
* 当 Agent 成功完成一个任务时，LLM 会将这段控制代码（如 `mineWood()`）保存到技能库中。后续遇到类似任务时，直接检索并运行该代码，无需再次调用 LLM，实现了技能的累积与复用。

### 3. 环境反馈与自我纠错（Environment Feedback & Self-Correction）
* 当 Action Agent 执行生成的 JavaScript 代码时，如果发生语法错误或游戏内逻辑失败（如：挖矿时被怪物打断、镐子坏了），Voyager 会将**代码报错信息（Traceback）**和**当前游戏状态**作为上下文重新喂给 LLM。
* LLM 根据报错自主修改代码，重新运行，直到任务成功。

## 关键贡献与对多智能体模拟的启示

* **终身学习范式**：证明了 Agent 可以在没有人类标注、没有预先训练奖励函数的情况下，通过与环境的高频交互，自主习得成百上千种复杂技能。
* **代码作为认知媒介**：将大模型的决策从“自然语言闲聊”提升到了“代码级精确控制”，极大提升了 Agent 在复杂 3D 实时环境中的生存能力。
* **对 3D 社会模拟的支撑**：Voyager 的技术直接启发了后续在《我的世界》等复杂 3D 沙盒中进行的大规模社会文明模拟（如 Project Sid），使得 Agent 能够进行真正的物理建造、资源采集和工具制造。
