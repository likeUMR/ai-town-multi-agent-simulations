## 核心架构：超大规模社交模拟引擎

OASIS 是一个专为模拟**社交媒体生态（如 Twitter/X、Reddit）**而设计的超大规模多智能体平台。为了支持高达 **100 万个 Agent** 的高并发、长周期交互，OASIS 重新设计了模拟器的底层架构：

```
┌────────────────────────────────────────────────────────┐
│                         OASIS                          │
│                                                        │
│  ┌───────────────────────┐   ┌──────────────────────┐  │
│  │  Environment Server   │   │ Recommendation System│  │
│  │ (Posts, Comments, DB) │   │ (Interest/Hot-score) │  │
│  └───────────▲───────────┘   └──────────▲───────────┘  │
│              │                          │              │
│              └─────────────┬────────────┘              │
│                            ▼                           │
│  ┌──────────────────────────────────────────────────┐  │
│  │                Scalable Inferencer               │  │
│  │              (vLLM / Parallel LLM)               │  │
│  └─────────────────────────▲────────────────────────┘  │
│                            │                           │
│  ┌─────────────────────────▼────────────────────────┐  │
│  │                    Agent Module                  │  │
│  │          (CAMEL Memory & 23 Social Actions)      │  │
│  └──────────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────────┘
```

### 1. 社交媒体环境服务器（Environment Server）
* 维护一个动态更新的社交网络拓扑图、帖子数据库和评论树。
* Agent 拥有高达 **23 种细粒度的社交动作空间**（包括：发帖、评论、回复评论、转发、点赞/踩、关注/取关、拉黑、静默等）。

### 2. 双重推荐系统（Recommendation System）
* 100万个 Agent 无法进行全连接的通信。OASIS 引入了与现实高度一致的推荐算法：
  * **兴趣推荐（Interest-based）**：基于 Agent 个人画像（Profile）与帖子内容的语义相似度进行个性化推送。
  * **热度推荐（Hot-score-based）**：基于帖子的点赞数、评论数和发布时间进行全网热门推送。

### 3. 高性能可扩展推理机（Scalable Inferencer）
* 传统的 Agent 框架（如 LangChain）在面对大规模并发时会因为 API 限制和单线程阻塞而崩溃。
* OASIS 底层集成了 **vLLM** 等高性能本地大模型推理框架，采用动态批处理（Dynamic Batching）和分布式 GPU 调度，使得百万级 Agent 的并发推理在工程上成为可能。

## 关键涌现现象：宏观舆论动力学

OASIS 成功复现了计算社会科学中极其难以通过传统 ABM 模拟的宏观社会现象：

1. **信息传播动力学（Information Spreading）**：
   * 模拟了谣言、突发新闻在社交网络中的传播曲线。研究发现，LLM Agent 展现出了与人类极其相似的“猎奇心理”和“情绪化转发”特征。

2. **群体极化（Group Polarization）**：
   * 在关于敏感议题（如气候变化、AI 伦理）的讨论中，受推荐系统“信息茧房”的影响，原本温和的 Agent 逐渐向极端立场靠拢，社区分裂为两个针锋相对的阵营。

3. **从众效应与羊群行为（Herd Effects）**：
   * 当某个帖子在初期获得大量点赞和正面评论时，后续刷到该帖子的 Agent 会倾向于无脑点赞或发表赞同意见，即使该帖子的内容与其初始画像存在轻微冲突。

## 学术与实用价值

* **计算社会科学的革命**：传统的社会学实验极难招募到成千上万的真实人类进行长期的社交媒体控制实验。OASIS 提供了一个可控、可重复、无伦理风险的“数字社会实验室”。
* **MAS 扩展定律（Scaling Laws）的探索**：首次系统性地研究了**智能体群体规模**对社会动力学的影响。实验证明，当 Agent 规模跨越某个临界点时，社会舆论的复杂度和涌现行为的真实度会发生显著的非线性跃升。
