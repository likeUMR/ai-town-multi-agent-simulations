## 核心架构：LLM 原生分布式模拟引擎

AgentSociety 是一个旨在模拟**真实城市物理空间与宏观社会经济运行**的超大规模平台。为了解决万级 Agent 在复杂城市环境中高频、长周期交互的算力与状态同步瓶颈，该平台引入了多项前沿技术：

```
┌────────────────────────────────────────────────────────┐
│                      AgentSociety                      │
│                                                        │
│  ┌──────────────────────────────────────────────────┐  │
│  │            Ray Distributed Computing             │  │
│  │     (Scales Agent Actors across GPU/CPU Nodes)   │  │
│  └────────────────────────▲─────────────────────────┘  │
│                           │                            │
│  ┌────────────────────────▼─────────────────────────┐  │
│  │               Flexible Environment               │  │
│  │       (Urban Spatial Map, Economy, Weather)      │  │
│  └────────────────────────▲─────────────────────────┘  │
│                           │                            │
│  ┌────────────────────────▼─────────────────────────┐  │
│  │                LLM-Native Agent Brain            │  │
│  │   (ReAct, Plan-Execute, MCP Tool Extensibility)  │  │
│  └──────────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────────┘
```

### 1. 基于 Ray 的分布式 Actor 架构
* 将每个 Agent 建模为一个独立的分布式 Actor。
* 利用 **Ray** 的分布式调度能力，将成千上万个 Agent Actor 动态分配到不同的计算节点（CPU/GPU）上运行，实现了极高的横向扩展性。

### 2. 模块化环境系统与 MCP 支持
* 模拟了逼真的城市物理与社会环境，包括：空间地图（住宅、写字楼、商场、街道）、动态经济系统（银行、企业、物价、税收）、天气与外部冲击。
* 原生支持 **Model Context Protocol (MCP)**，允许 Agent 动态加载和调用外部工具（如：搜索引擎、数据分析工具、地图导航），极大地扩展了 Agent 的行动边界。

### 3. 多重认知与决策模式
* 摒弃了单一的 Prompt 模板，支持 Agent 根据当前情境在多种推理模式之间自由切换：
  * **ReAct 模式**：用于快速的环境交互与工具调用。
  * **Plan-Execute 模式**：用于长周期的职业规划、消费理财决策。
  * **代码生成模式**：用于解决复杂的数理或工程计算。

## 核心应用：严肃的政策与社会实验沙盒

AgentSociety 部署了 **10,000+ 个 Agent**，模拟了超过 **500 万次** 的个体与环境交互。它被成功应用于五大计算社会学实验，且模拟结果与现实世界的数据高度对齐：

1. **全民基本收入（UBI）政策评估**：
   * 模拟政府向所有 Agent 定期发放无条件基本收入。
   * 观察发现：UBI 显著提升了低收入 Agent 的生活满意度，但自发导致了部分服务行业（如餐饮、清洁）的劳动力短缺，并引发了温和的通货膨胀。

2. **自然灾害（飓风）冲击下的群体行为**：
   * 模拟飓风袭击城市。
   * 观察 Agent 的逃生决策、物资抢购（囤积行为）以及邻里之间的互助/冲突行为，为城市应急管理提供了宝贵的演练数据。

3. **虚假/煽动性信息的社会传播**：
   * 模拟极端言论在城市居民网格中的扩散路径，评估不同社交媒体管制政策（如禁言、事实核查）的实际效果。

4. **群体极化现象的演化**：
   * 探究在不同推荐算法和线下社交圈下，城市居民在社会议题上的观点分裂过程。

5. **城市可持续性与通勤规划**：
   * 模拟居民的日常通勤（交通工具选择、路线规划），评估碳税、公交补贴等政策对城市整体碳排放的影响。

## 学术与现实意义

* **决策者的“水晶球”**：AgentSociety 证明了生成式社会模拟已经跨越了“娱乐沙盒”阶段，开始成为政府和企业评估宏观政策、预测社会风险、优化城市治理的**严肃决策支持工具**。
* **计算社会科学的集大成者**：将分布式计算、地理信息系统（GIS）、宏观经济学模型与大语言模型完美融合，确立了新一代数字孪生社会的技术标准。
