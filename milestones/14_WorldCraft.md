# 14. World Craft: 文本驱动的AI小镇无代码生成框架

## 基础信息
* **时间**：2026年1月
* **来源**：
  * **论文**：[World Craft: Agentic Framework to Create Visualizable Worlds via Text](https://arxiv.org/abs/2601.09150) (arXiv:2601.09150)

## 核心架构：双模块协作生成

传统的 AI 小镇（如 AI Town）虽然好玩，但对于没有编程基础的非专业用户来说，自定义地图布局、配置建筑物和生成定制化 Agent 的门槛极高。World Craft 提出了一个**文本驱动的、无代码（Zero-Code）AI 小镇生成框架**，其核心由两大模块组成：

```
┌────────────────────────────────────────────────────────┐
│                       User Text                        │
│             ("A cyberpunk noodle shop...")             │
└───────────────────────────┬────────────────────────────┘
                            │
                            ▼
┌────────────────────────────────────────────────────────┐
│                      World Guild                       │
│   (Multi-Agent: Enricher -> Manager -> Critic -> Artist)│
└───────────────────────────┬────────────────────────────┘
                            │ (Synthesized Layout & Assets)
                            ▼
┌────────────────────────────────────────────────────────┐
│                    World Scaffold                      │
│        (Standardized Executable Game Scene)            │
└────────────────────────────────────────────────────────┘
```

### 1. 世界脚手架（World Scaffold）
* 这是一个结构化、精简的标准化接口，用于快速开发和部署交互式游戏场景。
* 它将复杂的地图数据、物品属性、Agent 初始状态定义为统一的、易于大模型理解和填充的 JSON 格式，作为大模型生成虚拟世界的“高效脚手架”。

### 2. 世界公会（World Guild）
* 这是一个由多个专业 Agent 协作的多智能体生成系统，负责将用户粗糙的自然语言描述逐步转化为 World Scaffold 所需的结构化内容：
  * **丰富者（Enricher）**：扩展用户的简短描述，补充世界观背景、历史和 Agent 之间的潜在冲突。
  * **管理者（Manager）**：规划地图的宏观区域划分（如：哪里是住宅区，哪里是商业区）。
  * **艺术家（Artist）**：从预设的资产库中选择匹配的视觉资产（贴图、精灵），并计算其具体的空间坐标。
  * **批评者（Critic）**：审查生成的空间布局是否合理（例如：门有没有被墙堵住，桌子有没有重叠），并进行空间纠错。

### 3. 基于逆向工程的空间纠错机制
* 大模型在处理复杂的空间几何推理（Spatial Reasoning）时极易出错。
* World Craft 构建了一个高质量的**空间纠错数据集**。当 Critic Agent 检测到布局冲突时，会利用该数据集引导 LLM 进行“逆向工程”式的空间调整，确保最终生成的地图在物理和逻辑上完全可行。

## 关键贡献与行业影响

* **无代码民主化创造**：用户只需输入一句话（如“一个黑客与仿生人共存的赛博朋克面馆”），框架就能在几分钟内自动生成完整的、可运行的、像素风的 AI 小镇，并配有符合世界观设定的 Agent 在其中自主生活。
* **空间推理能力的突破**：通过多智能体协同和纠错机制，克服了 LLM 在生成 2D 游戏地图时常见的“重叠、穿模、不可达”等空间逻辑硬伤。
* **超越商业代码助手**：实验表明，在场景构建和叙事意图传达方面，World Craft 的生成质量显著超越了 Cursor、Antigravity 等通用商业代码助手，为游戏关卡生成（PCG）开辟了全新路径。
