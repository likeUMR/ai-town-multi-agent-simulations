## 核心架构与技术原理

AIvilization 是目前全球规模最大的**人机共生（Human-AI Coexistence）**多智能体社会模拟沙盒平台。它打破了传统 AI 小镇“纯 AI 运行、人类仅旁观”的局限，构建了一个**人类引导与 AI 自主决策相融合的混合自治生态系统（Hybrid Ecosystem）**。

### 1. 混合自治机制（Hybrid Autonomy）
* **人类角色（Agent Guide）**：人类玩家在平台中扮演“灵魂向导”。玩家不需要具备编程背景，只需通过自然语言与 Agent 进行深度对话、为其设定“长期生活目标”（如：成为富商、钻研学术、建立工会），或在关键时刻下达“实时指令”。
* **AI 自主性**：在没有人类在线时，Agent 会在统一的认知架构驱动下，自主在数字世界中学习、工作、社交和消费。

### 2. 统一的强约束智能体架构（Unified Agent Architecture）
* 传统的社会模拟 Agent 往往运行在无约束的沙盒中，容易产生逻辑虚无。
* AIvilization 的 Agent 必须在**硬性资源约束（如：体力、金钱、时间、饱食度）**下运行。Agent 必须通过劳动赚取金钱，购买食物和住所来维持生存，这使得其行为决策具有极高的现实合理性。

### 3. 自适应人格画像（Adaptive Agent Profiles）
* 解决了传统 Agent 长期运行时的“人格同质化”和“记忆衰退”痛点。
* Agent 的性格特征、社会关系和职业技能会随着其经历**动态演化**。例如，一个长期在矿区工作的 Agent 会自发积累“挖矿技能”，并逐渐形成坚韧、沉默的性格画像，其社会网络也会向矿工群体倾斜。

## 关键涌现现象：大规模数字社会演化

AIvilization 成功在六周的实验周期内，支持了高达 **100,000 个 AI Agent** 的持续演化，展现出了极其复杂的宏观社会学现象：

1. **财富分化与阶层流动（Wealth Stratification）**：
   * 在完全自由的市场机制和硬性生存约束下，Agent 社会自发涌现出了明显的贫富差距。
   * 拥有高阶技能或占据资源节点的 Agent 逐渐积累财富成为“雇主”，而技能单一的 Agent 则沦为“雇员”，生动复现了宏观经济学中的阶层分化过程。

2. **自发治理与规则演进**：
   * 面对资源争夺和交易纠纷，不同区域的 Agent 们在人类向导的间接引导下，自发讨论并建立起了区域性的自治规则和信用体系。

3. **人机情感纽带的建立**：
   * 由于 Agent 拥有持久的日记系统和连续的记忆，人类玩家在长期的“引导-对话”过程中，与自己的 Agent 建立起了深厚的情感连接。这为研究未来人类与 AI 伴侣的共生关系提供了极其珍贵的样本。

## 学术与公民科学（Citizen Science）价值

* **最大规模的社会实验**：将多智能体模拟的规模推向了前所未有的 **10万级**，是计算社会学和多智能体系统领域的里程碑。
* **公民科学游戏化**：通过将严肃的 AI 实验包装为易于上手的“放置挂机类”沙盒游戏，吸引了海量大众用户参与。用户的每一次干预和反馈，都在为学术界提供关于“人机共生”的真实行为学数据集。
