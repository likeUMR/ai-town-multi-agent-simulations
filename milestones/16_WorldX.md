# 16. WorldX: 一句话生成AI自主驱动的像素世界

## 基础信息
* **时间**：2026年4月
* **来源**：
  * **开源仓库**：[YGYOOO/WorldX](https://github.com/YGYOOO/WorldX)
  * **技术底座**：Phaser 3 (前端游戏引擎), Express + SQLite (后端服务)

---

## 核心架构：四模型角色协同驱动

WorldX 是开源社区中极具代表性的、能够**“一句话生成完整 AI 像素世界”**的端到端系统。它不仅能生成场景配置，还能利用图像生成模型和多模态模型，**从零开始绘制**出完全符合用户描述的、非模板拼接的地图美术和角色立绘。其系统由四个独立配置的模型角色协同运行：

```
┌────────────────────────────────────────────────────────┐
│                       User Prompt                      │
│          ("A wizard academy floating in sky")          │
└───────────────────────────┬────────────────────────────┘
                            │
                            ▼
┌────────────────────────────────────────────────────────┐
│                   Orchestrator Model                   │
│        (Designs World Structure, Characters, Rules)    │
└─────────────┬─────────────────────────────┬────────────┘
              │                             │
              ▼                             ▼
┌──────────────────────────┐   ┌─────────────────────────┐
│     Image Gen Model      │   │      Vision Model       │
│ (Generates Map & Sprites)│   │ (Reviews & Locates Map) │
└─────────────┬────────────┘   └────────────┬────────────┘
              │                             │
              └─────────────┬───────────────┘
                            ▼
┌────────────────────────────────────────────────────────┐
│                   Simulation Model                     │
│         (Drives Runtime Agent Behavior & Dialogues)    │
└────────────────────────────────────────────────────────┘
```

### 1. 编排引擎（Orchestrator）
* 负责解析用户的初始 Prompt，设计整个世界的宏观结构、区域划分、物理规则，并生成每个 AI 角色的姓名、性格、背景故事和初始人设。

### 2. 绘图模型（Image Gen）
* 拒绝使用死板的预设模板。该模型根据编排引擎的设计，实时生成全新的、完全符合特定世界观风格的地图美术资产（Map Art）和角色像素立绘（Sprites）。

### 3. 绘图审查（Vision）
* 这是一个多模态模型，负责审查 Image Gen 生成的地图质量，识别并精确定位地图中的关键区域、障碍物和可交互元素（如：哪里是门，哪里是椅子），将视觉图像转化为游戏引擎可读的物理碰撞网格。

### 4. 世界驱动引擎（Simulation）
* 负责驱动运行时的 Agent 行为。Agent 会感知周围环境，根据自身性格和记忆做出决策（移动、对话、使用物品），并支持跨越昼夜循环（Day/Night Cycles）的长期演化。

---

## 核心特色与上帝模式（God Mode）

* **真正的端到端无模板生成**：从世界观、地图美术、角色立绘到运行时的 AI 行为，全部由 AI 实时生成，实现了“输入一句话，收获一个活的世界”。
* **上帝模式（God Mode）**：
  * 允许人类用户作为“上帝”随时向全网广播突发事件（如“天降陨石”、“巨龙袭击”）。
  * 支持直接编辑任意 Agent 的记忆、人设，或者开启“架空对话沙盒”，观察世界线如何因为微小的干预而走向完全不同的分支。
* **多时间线系统（Timeline System）**：同一个初始世界可以孕育出多个不同的时间线分支，方便用户对比不同决策导致的社会演化结果。

---

## 关键贡献与行业启示

* **多模态生成在社会模拟的闭环**：WorldX 首次在开源社区实现了“文本 -> 图像 -> 视觉理解 -> 物理引擎网格 -> LLM 认知驱动”的完整闭环，代表了 AIGC 与多智能体模拟结合的最高工程水平之一。
* **未来游戏开发的雏形**：向人们展示了未来“生成式游戏（Generative Games）”的可能形态——游戏不再有预先写好的剧本和美术，一切都在玩家的一句话中实时诞生、实时演化。
